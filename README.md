# assets/

## founders.jpg

Foto de los fundadores del Método Lash Mastery. Se muestra en la pantalla
de bienvenida del quiz, debajo de la duración estimada y arriba del botón
"Comenzar".

**Specs:**
- Tamaño actual: 600×450 px, ~34 KB, JPEG baseline.
- Se muestra como un círculo de 132 px (mobile) / se ve bien en cualquier
  relación de aspecto cuadrada o 4:3.

## ¿Cómo reemplazarla?

Si quieres cambiar la foto:

1. Guarda tu nueva imagen como **`founders.jpg`** en esta misma carpeta.
2. Recomendado: 600×600 px (cuadrada) o 800×600 px (4:3), JPEG calidad 80–85%.
3. El HTML referencia `assets/founders.jpg` (ruta relativa), así que NO
   cambies el nombre del archivo.

Si por alguna razón la imagen no se carga (404, etc.), el sitio muestra
automáticamente un placeholder con las iniciales **"LM"** dentro del
círculo rosado. No se rompe el layout.
